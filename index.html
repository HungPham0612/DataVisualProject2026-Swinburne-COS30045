About This Project

Data Source

This visualization analyzes Australian road casualty data comparing statistics from 2011 and 2021. The dataset includes multiple dimensions: gender, age groups, road user types, counterparty types, and remoteness categories.

Methodology

All visualizations adhere to data visualization best practices including graphical integrity, accessibility standards (WCAG 2.1 AA), and responsive design principles. Interactive features enhance user understanding through tooltips, filtering, and dynamic annotations.

Technology Stack

  • D3.js v7 for data visualization
  • Responsive CSS Grid and Flexbox layout
  • Colorblind-safe color palettes
  • ARIA labels for screen reader accessibility
